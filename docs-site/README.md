# lawdit User Docs

This directory contains the Fumadocs-powered user documentation site for lawdit.

## Local Commands

```bash
npm install
npm run dev
npm run build
```

The root route redirects to `/docs`.

## Scope

The content is user-facing guidance for the prelaunch prototype. It must stay aligned with `../ACCEPTANCE.md`, `../docs/API_CONTRACT.md`, and the product design notes.

Do not use this docs site to claim legal advice, full GDPR compliance, production tenant access, production Microsoft Graph integration, or real deletion.
