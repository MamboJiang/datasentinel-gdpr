# Demo Script

## Demo Promise

DataSentinel does not just detect personal data. It makes every GDPR finding measurable, explainable, owner-routed, human-reviewed, and audit-ready.

## Three-Minute Flow

1. Open the admin dashboard and show source readiness.
2. Start a full scan on the controlled demo source.
3. Show scanned files, flagged files, data volume, progress, scan time, and review backlog.
4. Open a high-risk finding from the findings table.
5. Show redacted evidence, detector signals, context category, owner assignment, retention status, and risk explanation.
6. Record a human review decision with a reason.
7. Show the audit event created by that decision.
8. Run or show a delta scan that only processes changed files.
9. Open the evaluation tab and show accuracy, reproducibility, speed, and resource intensity.

## Required Demo Lines

- "Deletion is simulated in this prototype."
- "AI assists with classification and explanation, but the human reviewer has the last word."
- "Every review decision creates an audit event."
- "This is a governance loop, not a one-time scanner."
